About this project

As a Latin immigrant I’m aware of how effective communication can often get lost in translation. Knowing that San Francisco is home to a large Latin population, I felt compelled to ensure that language wouldn’t be a barrier for my community to find healthcare, especially during the year of 2020. So I built ¿Hablas Portuñol? - a Python web application that allows users to locate Spanish and Portuguese-speaking health providers in San Francisco. They can also write reviews, bookmark doctors and contribute to my open source data feature.

A quick overview of the process

I’m using PostgreSQL to store and manage all the data I gathered. Designing a database schema and then building effective queries was one of the biggest challenges of this project because when a user searches for a doctor or for a specialty, the server needs to check data from three different tables in the database before returning the search results.
To facilitate the communication between my Python application and the database, I’m using SQLAlchemy and then JavaScript to send network requests to the server, to load the map and its geolocation markers related to the search results. I’m also using JavaScript to create event listeners that add dynamic functionality to my front-end - such as the bookmark and unbookmark doctors button.
Besides using the MapBox API to render the map, I’m using its geocode API query to build a function that works as my own version of an API that parses addresses into coordinates. That was very useful to me since the data I gathered for the project and my crowdsource form feature are in street address format. Working with a crowdsource feature was also very challenging. I love the idea of building an open-source database of doctors with my community, but it is necessary to add constraints and validation to protect my users from fake information. My quick fix was to limit city and state, but I’m still working on implementing a more effective form of address validation and that’s something I’m really excited to learn more about.
I'm very thrilled about learning so much while building a useful resource of information. I hope many people can benefit from that!
